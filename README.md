# BrainNotes

## Descripcion:
Aplicación para tomar notas y gestionar conocimiento personal.

## Tecnologias:
- Python
- Tkinter
- SQite
- SQLAlchemy
- Pydantic

## Para ejecutar:
python main.py

## Motivación
Este proyecto surge de la necesidad de organizar notas y archivos que se encontraban dispersos
en el escritorio, así como de poder medir el tiempo invertido en cada tema.
A partir de esta necesidad real, el proyecto fue diseñado también como un ejercicio consciente
de buenas prácticas de arquitectura y separación de responsabilidades.
